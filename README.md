# #31 Contrast Checker

A hyper-minimalist tool to verify text readability and accessibility (WCAG).

## Core Philosophy
* **1 Tool / 1 Action**: Focuses exclusively on calculating color contrast ratios.
* **Direct Feedback**: The preview card changes color instantly to show "PASS", "WEAK", or "FAIL".
* **Visual Safety**: When it fails, the background turns light red to ensure the message is readable regardless of input.

## How to Use
1. Enter the **Background Color** (Hex code).
2. Enter the **Text Color** (Hex code).
3. Instantly see the Contrast Ratio and WCAG AA compliance status.
4. Use the **Clear (×)** buttons for quick resets.

## Technical Standard
* **WCAG AA Compliance**:
    * **PASS**: Ratio ≥ 4.5:1 (Standard text).
    * **WEAK**: Ratio ≥ 3:1 (Large text/UI components).
    * **FAIL**: Ratio < 3:1 (Not accessible).

## Privacy & Performance
* No login / No tracking / No ads / Browser-only.
* Optimized for low-spec PC and Mobile devices.
